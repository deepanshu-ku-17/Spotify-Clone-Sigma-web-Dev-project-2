# Spotify-Clone-Sigma-web-Dev-project-2
This project is a Spotify clone web UI clone created while learning through Sigma web development.

The goal of this project was to improve my understanding of modern website layouts, styling, and UI structure by recreating the look and feel of Spotify.

## Features

- Spotify-inspired interface
- Sidebar navigation
- Music cards and playlists
- Custom styling
- Hover effects
- Layout and alignment practice
- Clean UI structure

## Tech Stack

- HTML
- CSS
- JavaScript

## Learning Objectives

Through this project, I practiced:
- Flexbox and positioning
- Responsive layout concepts
- UI cloning techniques
- CSS styling and hover effects
- Basic DOM manipulation
- Structuring frontend projects

## Project Purpose

This project was built for learning and practice purposes only.

It helped me better understand how music streaming platform interfaces are designed and structured.

## Note

This is a frontend practice project and not a complete Spotify application.

Some features from the original platform are not implemented.
